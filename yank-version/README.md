# pleme-io · yank-version

> Polymorphic yank/unpublish — cargo yank / npm deprecate / pip remove. Surgical rollback for a single bad version (does NOT delete previous versions).

**Category**: `release-mgmt` — 📦 Release management — Changesets / semantic-release

**Auto-published**: pinnable via `@v0.13.x` tags or floating `@v1` / `@main`

## Auto-published on free public CI

Every push to `main` on `pleme-io/actions`:
1. `auto-bump.yml` fires (~10s) → tags `v0.13.{next}`
2. `release.yml` cuts the Docker image (if applicable) + fast-forwards `v1`
3. Consumers using `@v1` or `@v0.13.{x}` see the new revision automatically
